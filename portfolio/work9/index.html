Carolina Bustamante

Parlantes

Proyecto primer semestre 2016

Se realizo este modelo con el programa 3D Max ademas de un efecto con el cual se usaba un sonido como variable en la modelacion de este. El exterior esta hecho 100% de mdf, son multiples capas que finalmente forman el parlante. La parte interna es la misma que se podria encontrar en cualquier otro equipo de sonido.

Cada uno tiene una dimencion de 15x15x18cm
